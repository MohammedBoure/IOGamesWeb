# Neon Aim Arena Models

Runtime-ready 3D models for the shooter game.

- `characters/swat.gltf`: character model candidate for remote players.
- `weapons/low-poly-guns/*.gltf`: self-contained weapon models suitable for browser loading.

Only the runtime-ready files needed by the game are kept here. Verify the asset licenses before public redistribution if these files did not come from a clearly licensed source.
