# Neon Aim Arena Models

Runtime-ready 3D models for the shooter game.

- `characters/ninja.glb`: character model candidate for remote players.
- `weapons/low-poly-guns/*.gltf`: self-contained weapon models suitable for browser loading.

The original multi-format gun source pack is kept locally under `asset-sources/` and ignored by Git. Verify the asset licenses before public redistribution if these files did not come from a clearly licensed source.
