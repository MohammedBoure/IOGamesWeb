# World Model Assets

Selected runtime-friendly world props are kept here as `OBJ` + `MTL` pairs.

## Packs

- `farm`: farm buildings and landmarks suitable for shooter cover, arenas, and racing track scenery.
- `nature`: trees, plants, rocks, and small cover props suitable for both shooter and racing maps.

## Source Selection

Only lightweight browser-ready `OBJ`/`MTL` files were retained. Heavy source formats such as `.blend` and `.fbx`, plus unused variants, were deleted after selection to keep the repository small.
